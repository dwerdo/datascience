DataScience's Frontend Code Test
==================
## Info
Code Test was done with React.js

## Dependencies
Bower
- jquery
- react
- bootstrap

NPM
- http-server

## Installation 
If you have bower installed you can  run:
```
bower install
```
Install a simple http server:
```
npm install http-server -g
```
run:

```
http-server
```
